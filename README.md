# DevOps Hackathon Usecase - Complete Project

This repository contains a sample healthcare microservices app (Patient, Appointment, Order)
plus Terraform infrastructure, Kubernetes manifests, and GitHub Actions workflows for CI/CD.

See folders:
- application-service/: patient-service (Node), appointment-service (Node)
- order-service/: Java Spring Boot service
- k8s/: Kubernetes manifests
- terraform/: Terraform modules and envs
- .github/workflows/: CI and terraform pipelines

Follow the root README in /docs for detailed installation steps.
